# ADA Event Studio

## プロジェクトの目的

ADAが開催するイベントを支援する統合イベントシステムを開発する。

対象イベント

- オークション
- 抽選会
- 表彰式
- クイズ
- 来場者受付

---

## 開発方針

- シンプルで分かりやすいUI
- イベント中に止まらない設計
- Firestoreによるリアルタイム同期
- iPad・PC・大型スクリーンで利用可能
- 機能追加しやすい構造を維持する

---
