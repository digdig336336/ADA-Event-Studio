ADA Event Studio v1.0
1. 要件定義書（requirements.md）
目的

イベント運営を円滑に行うため、オークション・抽選・表彰を一元管理できるシステムを構築する。

最終的にはiPadから操作し、大型スクリーンへリアルタイム表示できるシステムを目指す。

開発目的
オークション進行の効率化
操作ミスの削減
イベント運営の標準化
イベント履歴の保存
Firestoreによるリアルタイム同期
将来的な複数端末対応

NAFES
ADAイベント
チャリティーオークション
